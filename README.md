# AI Smart Calculator (NLP-Based)

This project is an AI-powered smart calculator that understands
natural language mathematical expressions such as:

- "two plus three"
- "20 percent profit on cost 500"
- "10 percent loss on cost 300"

## Features
- Natural Language Processing (NLP)
- Word-to-number conversion
- Business calculations (profit, loss, percentage)
- GUI using Tkinter
- Error handling

## Technologies
- Python
- Tkinter
- NLP (rule-based)
- AI logic

## Use Case
- Educational learning
- SME business decision support
- AI & ML academic projects

## How to Run
```bash
python smart_calculator.py
